### ğŸŸ¤ ë¬¸ì œ í’€ì´ (Algorithm)

```java
class Solution {
  public int removeDuplicates(int[] nums) {

    int index = 0;
	
    int tempValue = Integer.MIN_VALUE;
    int count = 0;

    int result = 0;

    for (int i = 0; i < nums.length; i++) {
      if (nums[i] == tempValue) {
        count++;
      } else {
        tempValue = nums[i];
        count = 0;
      }

      if (count < 2) {
        nums[index++] = nums[i];
        result++;
      }
    }

    return result;
  }
}
```
